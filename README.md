FixFirst

📌Overview

FixFirst is a citizen-first civic-tech platform designed to address delays, lack of transparency, and limited accountability in public issue resolution.

It enables users to report civic issues with precise location details, track their status, and contribute to community-driven validation. The platform focuses on improving efficiency, reducing duplicate complaints, and enhancing trust between citizens and authorities.

🔵Features

Precision Reporting
Users can report issues by selecting an exact location on the map or entering it manually. Location data can also be captured automatically for accuracy.

Image Capture Support
Users can attach real-time images using their device camera to provide visual proof of issues.

Live Status Tracking (Simulated)
Issues are tracked through stages such as Open, In Progress, and Resolved to demonstrate real-time monitoring in a production-ready system.

Community Validation & Smart Prioritization
Issues are validated and prioritized based on community input, including upvotes, location density, and severity.

Duplicate Issue Reduction
Multiple reports for the same issue are consolidated to avoid redundancy and improve efficiency.

🔵Role-Based Access System

Citizen Dashboard: Report issues and upvote existing ones

Authority Dashboard: View issue statistics, monitor progress, and manage resolution

Local Data Handling
A local database is used to store and fetch issue data, including updates such as upvotes and status change.
